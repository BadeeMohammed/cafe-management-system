This is a basic cafe management system made by python.
In this, the program asks you to order something.. and places an order. Again it asks if you want anything else.
Atlast this program gives a message if the order is placed successfully and generates a final bill to pay.
The best part is that the hotel owner/manager can use this program because, whenever this program is runned, automatically a file named sales_record is created.
The file stores the number of orders placed and the earning generated by that order!
This program is very useful for tracking orders and earning. We can also include profit and loss in it. Which really helps the manager and the owner to track the expenses and take next step in the business.
In this program I have used While loop, Dictionary, file modes, conditional statements etc.
