This is a basic cafe management system made by python.
In this system, the program asks you to order something and places an order. Again it asks if you want anything else.
Atlast this program gives a message that the order is placed successfully and generates a final bill to pay.
The best part is that the hotel owner/manager can use this program because, whenever this program is run, automatically a file named sales_record is created.
The file stores the number of orders placed and the earning generated by that order!
This program is very useful for tracking orders and earning. We can also include profit and loss in it. Which really helps the manager and the owner to track the expenses and take next step according to it.
In this program I have used While loop to ask again and what if you want anything until you say "no".
Dictionary to save the price of food with their key pairs.
File modes to create and open a file and record the earning and number of orders.
Conditional statements to perform task according to user's will.
